# Tetris Real Blocks

Web app mobile en français.

## Contenu
- écran de jeu séparé
- manette tactile indépendante
- score et record
- 10 niveaux
- 10 décors générés localement
- jouable sur navigateur mobile ou PC

## Lancer
Ouvrir `index.html` dans un navigateur.

## Mise en ligne
Déposer le dossier sur GitHub Pages, Netlify ou Vercel.
